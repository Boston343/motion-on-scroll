## Minimal bootstrapping

You'll need to import MOS styles, and run the `MOS.init()` function.


<Tabs>
  <TabItem label="Vanilla JS">

```html
<!-- Import MOS styles -->
<link rel="stylesheet" href="https://unpkg.com/motion-on-scroll/dist/mos.css" />

<!-- Initialize MOS -->
<script type="module">
  import { MOS } from 'motion-on-scroll';
  MOS.init();
</script>

<!-- Somewhere in your markup -->
<section data-mos="fade-up">
  ...
</section>
```
  </TabItem>
  <TabItem label="Astro">

```astro
---
import 'motion-on-scroll/dist/mos.css';
---
<html lang="en">
  <head>

```
  </TabItem>

### CSS file

MOS ships with a small stylesheet that ensures elements are hidden upon initial page load. Import it from `motion-on-scroll/dist/mos.css` or copy the file into your own pipeline. The contents of the file are copied below for convenience.

```css
/* no pointer events until animation starts */
[data-mos] {
  pointer-events: none;
}

[data-mos].mos-animate {
  pointer-events: auto;
}

/* fade & zoom presets */
[data-mos^="fade"],
[data-mos^="zoom"] {
  will-change: opacity, transform;
  opacity: 0;
}

/* slide presets */
[data-mos^="slide"] {
  will-change: transform;
  visibility: hidden;
}
[data-mos^="slide"].mos-animate {
  visibility: visible;
}

/* flip presets */
[data-mos^="flip"] {
  will-change: transform;
  backface-visibility: hidden;
}
```
